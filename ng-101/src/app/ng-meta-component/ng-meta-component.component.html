About Angular Components

Use the ng cli to generate a component via ng generate component and then reference the component from another component or module

Component vs Module

Component
A component references 0 or more other components
Module
A module bootstraps a component
Root Component
The root component can include additional components
View
A view is the combination of a component and a template
Host View
The components with it's default template defines the host view
View Hierarchy
When a component can contains one or more views, this is called a View Hierarchy
