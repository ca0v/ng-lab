Injector is a sort of factory, creating providers that implement a spec

Providers are used to inject dependencies

At the bottom of the form you should see a ticking clock, which uses SystemTimeService.getTime().

The dictionary.module registers the injectable SystemTimeService this as a provider.

The dictionary.component uses/injects this provider via declaring a constructor parameter.

By decorating the @Injectable with providedIn: 'root' it becomes globally available, and I could have skipped the registration in the dictionary.module.
